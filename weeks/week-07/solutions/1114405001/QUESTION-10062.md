## 題目敘述

有 N 頭（2 ≤ N ≤ 80,000）乳牛，每頭乳牛都有一個在 1 到 N 範圍內的獨特編號。
牠們在晚餐前去附近的「飲水站」喝了太多啤酒，一時判斷失準。
排隊吃晚飯時，牠們沒有依照編號從小到大的正確順序排列。
不幸的是，農夫 FJ 無法直接幫牠們排序，而且他的觀察能力也不太好。
他沒有記錄每頭乳牛的編號，而是統計了一個奇怪的數據：對於隊伍中的每頭乳牛，他知道在該乳牛前面、編號比它小的乳牛有幾頭。
請根據這份資料，告訴 FJ 乳牛的正確排列順序。

## 輸入說明

- 第 1 行：一個整數 N。
- 第 2 至 N 行：共 N-1 行，每行描述該位置的乳牛前面、編號比它小的乳牛數量。
  第一頭乳牛前面沒有任何乳牛，因此不列出。
  第 2 行描述第 2 個位置的乳牛前面編號較小的牛數；第 3 行描述第 3 個位置的情形，以此類推。

## 輸出說明

- 共 N 行，每行輸出該位置乳牛的編號。
  第 1 行為隊伍第 1 個位置的乳牛編號，第 2 行為第 2 個位置，以此類推。

---

## 解題思路

**從右到左還原**：

對於隊伍中第 `i` 個位置的牛，已知其前面有 `k[i]` 頭編號比它小的牛。
若從最後一個位置往前處理，每次在「尚未分配的編號」中找第 `k[i]+1` 小的那個，即為該位置的牛編號，然後將其從可用集合中移除。

**為何正確**：處理第 `i` 頭牛時，位置 `i+1` 到 `N` 已確定，剩餘未分配的編號即為位置 `1` 到 `i` 的牛所屬。
由於前面恰好有 `k[i]` 個比它小，它在剩餘編號中就是第 `k[i]+1` 小。

**資料結構**：使用 Fenwick Tree（BIT）維護可用編號，支援：
- 第 k 小查詢：二分搜 + 前綴和，O(log²N)
- 刪除元素：O(log N)

總時間複雜度：O(N log² N)

## 解題代碼

```python
import sys
input = sys.stdin.readline

def main():
    N = int(input())
    k = [0] * (N + 1)
    for i in range(2, N + 1):
        k[i] = int(input())

    # Fenwick Tree
    bit = [0] * (N + 2)

    def update(i, delta):
        while i <= N:
            bit[i] += delta
            i += i & (-i)

    def query(i):
        s = 0
        while i > 0:
            s += bit[i]
            i -= i & (-i)
        return s

    # 找第 kth 小（1-indexed）
    def find_kth(kth):
        lo, hi = 1, N
        while lo < hi:
            mid = (lo + hi) // 2
            if query(mid) >= kth:
                hi = mid
            else:
                lo = mid + 1
        return lo

    # 初始化：所有編號 1..N 皆可用
    for i in range(1, N + 1):
        update(i, 1)

    result = [0] * (N + 1)
    for i in range(N, 0, -1):
        rank = k[i] + 1          # 在剩餘可用編號中的排名
        num = find_kth(rank)
        result[i] = num
        update(num, -1)          # 從可用集合移除

    print('\n'.join(map(str, result[1:])))

main()
```

## 測試用例

**輸入**：
```
5
1
2
1
0
```

**預期輸出**：
```
2
4
5
3
1
```

**驗證過程**（可用編號從 [1,2,3,4,5] 開始，從右到左）：
- i=5: k=0 → 第1小 = **1**，剩餘 [2,3,4,5]
- i=4: k=1 → 第2小 = **3**，剩餘 [2,4,5]
- i=3: k=2 → 第3小 = **5**，剩餘 [2,4]
- i=2: k=1 → 第2小 = **4**，剩餘 [2]
- i=1: k=0 → 第1小 = **2**，剩餘 []

結果：[2, 4, 5, 3, 1] ✓
