## 題目敘述

地球半徑為 **6440 公里**。若兩顆衛星與**地球圓心**形成一個角度，請計算兩顆衛星之間的**弧長**和**弦長**距離。

兩顆衛星在**同一軌道**上（圓形軌道，非橢圓形）運行。

**計算方式：**
- 設衛星距地表高度為 **s**，則衛星到地心的距離（半徑） r = 6440 + s
- 設兩衛星與地心所成夾角為 **a**（弧度）
- **弧長** = r × a
- **弦長** = 2 × r × sin(a / 2)

## 輸入說明

- 輸入包含多組測試資料，每組一行。
- 每行包含兩個整數 **s**（衛星距地表高度，公里）和 **a**（夾角），以及一個字串 `'min'` 或 `'deg'`：
  - `deg`：角度以**度（degree）** 表示
  - `min`：角度以**分（arcminute）** 表示（1 度 = 60 分）

## 輸出說明

每組測試資料輸出一行，包含兩個浮點數：
1. **弧長**（公里）
2. **弦長**（公里）

兩數均保留**小數點後六位**。

---

## 解題思路

1. **參數解析**：
   - 讀取衛星高度 s（公里）
   - 讀取夾角 a 和單位（'deg' 或 'min'）
   - 若是 'min'，先轉換為度：度 = 分 / 60

2. **單位轉換**：
   - 將角度從度轉換為弧度：弧度 = 度 × π / 180

3. **半徑計算**：
   - 衛星到地心距離：r = 6440 + s

4. **距離公式**：
   - **弧長** = r × a（a為弧度）
   - **弦長** = 2 × r × sin(a / 2)

5. **輸出格式**：
   - 輸出兩個浮點數，各保留小數點後 6 位

## 解題代碼

```python
import math

while True:
    try:
        line = input().split()
        s = int(line[0])           # 高度
        angle_value = float(line[1])  # 角度值
        unit = line[2]             # 'deg' 或 'min'
        
        # 轉換為度
        if unit == 'min':
            angle_degree = angle_value / 60
        else:  # 'deg'
            angle_degree = angle_value
        
        # 轉換為弧度
        angle_radian = angle_degree * math.pi / 180
        
        # 衛星到地心的距離
        r = 6440 + s
        
        # 計算弧長和弦長
        arc_length = r * angle_radian
        chord_length = 2 * r * math.sin(angle_radian / 2)
        
        # 輸出結果
        print(f"{arc_length:.6f} {chord_length:.6f}")
    
    except EOFError:
        break
```

**驗證計算**：

```
測試 1: s=500, a=30 deg
- r = 6440 + 500 = 6940
- 弧度 = 30 × π/180 = π/6 ≈ 0.523599
- 弧長 = 6940 × 0.523599 ≈ 3633.775503
- 弦長 = 2 × 6940 × sin(π/12) ≈ 3592.408346 ✓

測試 2: s=700, a=60 min
- r = 6440 + 700 = 7140
- 60分 = 1度 = π/180
- 弧長 = 7140 × (π/180) ≈ 124.616509
- 弦長 = 2 × 7140 × sin(π/360) ≈ 124.614927 ✓
```
